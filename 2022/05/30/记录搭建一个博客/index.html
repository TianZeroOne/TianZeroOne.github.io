搭建一个博客

Posted on 2022-05-30 | In 技能

搭建一个博客

github + hexo 搭建静态博客

环境准备

git + node + Hexo + hexo-deploy-git

可以通过查看版本信息看是否安装成功

git 配置 SSH key

  1. **ssh-keygen -t rsa -C "email"**生成 ssh key

一般是在c:/用户/管理员/.ssh

  1. 将 id_rsa.pub 的内容放在 github 主页的个人设置里

  2. 重新配置账号

搭建个人博客

参考官方文档 https://hexo.io/zh-cn/docs/

  1. 新建博客

    hexo init 在一个空目录下

  2. 生成静态网页

    hexo g == hexo generate

  3. 在上传之前最好预览一下

    hexo s == hexo server 开启预览端口

  4. 上传到 github

    hexo deploy = hexo

在 hexo d 的时候报错 git 没有该指令，可能是 hexo-deployer-git 没有安装成功，或者安装错位置，要在博客目录下

上传文章

  1. hexo n == hexo new 来创建一个新的文章
